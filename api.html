    API

    Voor de communicatie tussen de logische en toepassingen laag heb je een interface nodig. Dit interface is de API. Voorbeelden hiervan zijn Android API, Windows API en IOS SDK. Apps worden gemaakt voor bepaalde software API's zorgen voor de consistentie voor alle apparaten. Ook zorgt de API voor bepaalde functies van de telefoon.
